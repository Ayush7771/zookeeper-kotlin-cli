# 🦁 Zookeeper

Zookeeper is a simple Kotlin CLI-based application that allows users to explore different animal habitats by entering an index number. This project is designed to strengthen fundamental programming concepts like loops, arrays, and user input handling.

---

## 🚀 Features

* 🐪 View different animal habitats (Camel, Lion, Deer, Goose, Bat, Rabbit)
* 🔢 Select habitats using index-based input
* ❌ Handles invalid inputs gracefully
* 🔁 Runs continuously until the user types `"exit"`

---

## 🛠️ Technologies Used

* Kotlin
* CLI (Command Line Interface)

---

## 📸 How It Works

1. The program prompts the user to enter a number (0–5)
2. Based on the input, it displays the corresponding animal habitat
3. If the input is invalid, an error message is shown
4. The program continues running until `"exit"` is entered

---

## ▶️ Example

```
Please enter the number of the habitat you would like to view: 2
[Deer habitat appears]

Please enter the number of the habitat you would like to view: exit
See you later!
```
